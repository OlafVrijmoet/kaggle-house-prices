# Data

The competition data belongs to Kaggle and cannot be redistributed here.

Download it with the Kaggle CLI (requires a Kaggle account and an API token in `~/.kaggle/kaggle.json`):

```
kaggle competitions download -c house-prices-advanced-regression-techniques -p data --unzip
```

That places `train.csv`, `test.csv`, `sample_submission.csv` and `data_description.txt` in this folder, which is where the notebooks expect them.
